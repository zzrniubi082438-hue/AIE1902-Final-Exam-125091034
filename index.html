Model Details
Ensemble (Average of 3 Models)
ŷ(t+1) = ( ŷ_RW(t+1) + ŷ_AR(t+1) + ŷ_EWMA(t+1) ) / 3
1) Random Walk with Drift
y(t+1) = y(t) + μ μ = mean( y(t) - y(t-1) ) over the last L1 differences
2) AR(1)
y(t) = φ · y(t-1) φ is estimated by OLS slope (through origin) using the last L2 pairs ŷ_AR(t+1) = φ · y(t)
3) EWMA
ŷ(t+1) = α · y(t) + (1-α) · ŷ(t) initialize: ŷ(1) = y(1)
